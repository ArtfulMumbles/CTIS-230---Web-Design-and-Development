My Thoughts on TFT Set 5

Initial Release:

Though the new Set for Teamfight Tactics has not yet been released on public servers, it is currently available through the PBE. The launch can so far be described as smooth, and there have been no large bugs as of yet. At the time, it is unclear when a live server release can be expected, however estimates put it to be within the next week.

Changes from Set 4:

There have been several important changes between this set and the last. For starters, the chosen mechanic will not be returning this set. From my perspective, this is a welcomed change. Chosen in the previous set were unweidly and often times unnessessary. Carry champs of the last set, more below, could do fine without a chosen. In addition, max rank perks that required a chosen often also required a spatula to complete, making the aspect of randomly getting a chosen completely irrelevant. Even if a spatula was not required, it was not particularly hard to force a particular chosen by rerolling or simply by being patient.

What I am Looking for From Set 5:

Last set was a complete disappointment in my mind. While there were new features, the game could not have become more bland. The presence of super carry champs like Kayle and Aurelion Sol meant that certain comps could never be viable. Set 4 was not about making a team of champions, but rather about buffing one specific champion that had the ability to carry. While this has been relatively for all other sets, it was especially prevalent last Set. From Set 5, I am hoping that we there will be new avenues to victory that involve carefully manipulating your economy and your shop, and that the reliably forcing the same comp every game will no longer be an option.
